# ScoreMaster

Приложения для подсчета очков в игре в карты.

# Установка

1. Скопируйте репозиторий.
2. Выполните команду `flutter pub get`.

# Сборка приложения

1. Выполните команду `flutter build apk` или `flutter build apk --split-per-abi`.
2. Выполните команду `flutter install`.

# Сборка релизного APK (или AAB):

1. Выполните команду: `flutter build apk --release` для сборки APK, или Выполните команду: `flutter build apk --release` для сборки APK, или `flutter build appbundle --release` для сборки AAB для сборки AAB.
2. Установка на устройство:
  - Подключите Android-телефон к компьютеру через USB кабель. Убедитесь, что на телефоне включена отладка по USB в настройках разработчика.
  - Найдите сгенерированный APK файл на компьютере и скопируйте его на телефон в любую удобную директорию.
  - На телефоне, откройте файловый менеджер, найдите скопированный APK файл и нажмите на него. Следуйте инструкциям на экране для установки приложения.
  - Если у вас установлен **Flutter SDK** и **Android Debug Bridge (adb)**, вы можете использовать команду `adb install путь_к_вашему_apk.apk` для установки.
