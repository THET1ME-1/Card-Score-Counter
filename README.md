# Приложение ScoreMaster

Приложение для отслеживания очков в игре «КунКен» с функциями учета раундов, управления игроками, истории сессий, статистики побед и достижений (в разработке).

# Установка

1. Скопируйте репозиторий.
2. Выполните команду `flutter pub get` для установки всех необходимых пакетов для работы.

# Сборка релизного APK (или AAB):

1. Выполните команду: `flutter build apk --release` для сборки APK, или `flutter build appbundle --release` для сборки AAB для сборки AAB.
2. Выполните команду `flutter install` для установки приложения на телефон.
