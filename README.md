# ScoreMaster — счётчик очков для игры «КунКен»

Приложение на Flutter для отслеживания очков в карточной игре «КунКен» (Conquian):
учёт раундов, управление игроками с аватарками, история сессий, статистика побед
и таблица лидеров.

## Возможности

- **Игроки** — создание профилей с именем, цветом и фото; редактирование и удаление.
- **Табло** — пораундовый ввод очков, автоматическое выбывание при 101 очке,
  отмена и правка последнего раунда, подсветка текущего игрока.
- **История игр** — сохранение всех партий, продолжение незавершённых, сортировка по дате.
- **Статистика** — победы по дням/месяцам/годам, процент побед, графики (fl_chart).
- **Лидеры** — рейтинг по победам с медалями и круговой диаграммой.
- **Резервные копии** — экспорт и импорт всех данных в JSON (раздел «Настройки»).
- **Темы** — Material 3, светлая и тёмная темы, настройка размера текста.

## Архитектура

- `lib/models/` — модели данных (`GameSession`, `PlayerProfile`).
- `lib/services/game_repository.dart` — единый слой доступа к данным (поверх `shared_preferences`).
- `lib/theme/app_theme.dart` — централизованные темы Material 3.
- `lib/*_screen.dart` — экраны приложения.

## Установка

```bash
flutter pub get
```

## Сборка релизного APK / AAB

Проект сейчас задан как Flutter-модуль (есть только `.android`/`.ios`). Чтобы собрать
обычное приложение, один раз сгенерируйте платформенные папки:

```bash
flutter create --platforms=android,ios --org com --project-name score_master .
```

После этого:

```bash
flutter build apk --release        # APK
flutter build appbundle --release  # AAB
flutter install                    # установка на подключённое устройство
```
